# Beej's Guides

This repository has examples from the [Beej guides](https://beej.us/guide/). Well, the ones related to C, rather.

## Beej's Guide to C - Tutorial

Found under [c-tutorial](./c-tutorial/) in the repository root, and are working examples from the book [Beej's Guide to C â€” Tutorial](https://beej.us/guide/bgc/).

### Build

To compile all examples:

```bash
git clone https://github.com/juliojimenez/beej
cd beej
cd c-tutorial
make
```

Clean up:

```bash
make clean
```
